Smart Translator Chrome 扩展发布指南

1. 准备工作

1.1 开发者账号注册

  • 访问 Chrome Web Store Developer Dashboard
  • 支付一次性注册费（$5.00 USD）
  • 完成开发者账号验证

1.2 准备发布材料

确保以下文件已准备就绪：

extension/
├── manifest.json
├── content.js
├── background.js
├── config.js
├── content.css
├── icons/
│   ├── icon16.png
│   ├── icon32.png
│   ├── icon48.png
│   └── icon128.png
├── README.md
└── LICENSE

1.3 准备商店展示材料

  • 商店图标（128x128 PNG）
  • 至少一张截图（1280x800 或 640x400）
  • 简短描述（最多132字）
  • 详细描述（最多16,000字）
  • 宣传图片（440x280，可选）
  • 宣传视频（YouTube，可选）
注意： 所有图片必须是 PNG 或 JPEG 格式，大小不超过 2MB。

2. 打包扩展

2.1 创建发布包

执行以下命令创建 ZIP 包：

zip -r smart-translator.zip extension/* -x "*.git*" -x "*.DS_Store"
提示： Windows 用户可以使用 7-Zip 或其他压缩工具创建 ZIP 文件。

3. 提交审核

3.1 基本信息填写

  • 扩展名称：Smart Translator - 智能翻译
  • 简短描述：优雅的英文到中文翻译工具，支持智能缓存、上下文感知和现代化UI设计。
  • 详细描述：（见下文）

3.2 详细描述模板

Smart Translator 是一款现代化的翻译扩展，让英文阅读更轻松：

主要特点：
✓ 即时翻译：选中即译，无需额外点击
✓ 智能缓存：相同内容秒速翻译
✓ 优雅界面：现代化设计，支持暗黑模式
✓ 智能分段：自动识别文本结构
✓ 隐私保护：本地缓存，无需登录

使用方法：
1. 安装后即可使用，无需配置
2. 选中任意英文文本自动显示翻译
3. 支持长文本智能分段
4. 右上角按钮切换暗黑模式

隐私说明：
- 仅在必要时访问翻译API
- 所有缓存均存储在本地
- 不收集任何用户数据

3.3 类别与可见性

  • 主类别：生产力工具
  • 子类别：翻译工具
  • 语言：中文（简体）
  • 可见性：公开
